# Whisper GDExtension

## Developing

### Requirements

- Python 3.11+
- SCons `pip install SCons`

### Setup

Clone a local copy of godot-cpp or add a symlink to the godot-cpp folder in the root of this project.

```bash
git clone --depth 1 --branch 4.5 https://github.com/godotengine/godot-cpp.git godot-cpp
```
